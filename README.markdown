Traveller Campaign Binder
=========================

![A spaceship](roll20-banner.jpg?raw=true)

This repo contains ideas and plans for a Traveller campaign I'll be
starting later this year.

Ideas for content
-----------------

- Lines / veils / tone / genre
- Campaign state:
  - PCs
  - Major NPCs
  - Factions
  - Plot points
- Resources
  - Reference material
  - Names
  - Rules summaries: rulebook excerpts & references, cheatsheets (etc)
  - Random tables (events, rumours, gossip, hooks, events)
- Session log
  - Timeline / index
  - Per-session:
    - Prep
    - Notes
